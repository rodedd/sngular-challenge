# SNGULAR Challenge

### About
Technical challenge for the interview with SNGULAR.
 
### Built with

* **Library**: React.js
* **Styling**: CSS Modules
* **Bundler**: Vite
* **Testing**: Vitest

### Scripts
* **Install dependencies**
  ```console
  npm install
  ```
* **Run on local server**
  ```console
  npm run dev
  ```
* **Run tests**
  ```console
  npm run test
  ```
